# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitHub Actions CI pipeline for automated testing
- ESLint configuration with TypeScript and React support
- Professional open-source documentation (CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md)
- Issue and PR templates for better community contributions
- Branch protection on `main` requiring PRs and CI checks

### Changed
- Development workflow now uses `develop` branch for integration

## [0.1.0] - 2024-XX-XX

### Added
- Initial release of On-Chain Disc Golf PWA
- Nostr-based decentralized scorekeeping
- Cashu eCash wallet integration
- NWC (Nostr Wallet Connect) support
- Round creation and management
- Multi-player scoring
- Round history and statistics
- QR code sharing for round invites
